반복문 사용하기

반목문이란 주어진 조건에 따라서 문장을 반복 실행하게 해준다.
자바 스크립트에서 반복문은 for문, while문, do while문이 있다

 

FOR문

for (초기값; 최종값; 증가식)

 

사용예) 특정 문자열을 400번 반복해서 화면에 출력

    <script>
        for(i=1; i<=400; i++)
        document.write("사랑해")
    </script>

FOR문(중첩for문)

사용예) 구구단 출력

    <script>
        for(var i=2; i<=9; i++){
            document.write("<h1>"+i+"단 </h1> ")

            for(var k=1; k<=9; k++){
                document.write(i + "x" + k + "=" + i*k, "<br />")
            }
        }
    </script>

 

WHILE문

while(조건문)
{
     A
}
     B

// 조건문이 참이면 A를 실행하고 조건문이 거짓이면 B를 실행

 

사용예) 1부터 10까지의 합 계산

    <script>
        x=0
        total=0

    while(x<10)
    {
        x++
        total=total+x

        document.write(x)

        if(x!=10)
        document.write("+")
    }

        document.write("="+ total)
    </script>

DO WHILE문

WHILE문은 조건을 확인한 후에 문장을 실행하지만, DO WHILE문은 실행한 다음 조건을 확인하도록 해주는 반복문이다. 그러므로 DO WHILE문은 조건에 상관없이 반복 문장을 최소 한번은 실행하게 된다.

do
{ A }
while (조건)
B

 

사용예) 1부터 10까지의 합 계산

    <script>
        x=0
        total=0

        do
        {
            x++
            total=total+x

            document.write(x)
            if(x!=10)document.write("+")
        }

        while(x<10)
        document.write("="+ total)
    </script>

 

CONTINUE문

반복문 문장 안에 사용하여 반복 실행 중에 조건문으로 되돌아 가게 해주는 역활을 한다.
즉, 반목문에서 특정 부분을 실행하지 않게 하고 싶을 때 간단하게 사용할 수 있다.

사용예) 1부터 10까지의 수중에서 3을 제외한 수의 합을 계산

    <script>
        x=0
        total=0

        while(x<10)
        {
            x++

            if(x==3) continue
            total=total+x
            document.write(x)

            if(x!=10)
            document.write("+")
        }
        
        document.write("="+ total)
    </script>
