Math.속성 : Math 객체

수학에서 사용하는 특수함수나 특수값을 사용할 수 있게 해준다.
특수함수란 지수함수, 삼각함수, 로그함수등을 말한다.

E : 오일러 상수 약 2.718
LN10 : 10의 자연 로그 값 약 2.302
LN2 : 2의 자연 로그 값 약 0.693
LOG10E : 밑이 10인 E 로그 약 0.434
LOG2E : 밑이 2인 로그 약 1.442
PI : 원주율 약 3.14159
SQRT1_2 : 1/2의 제곱근 SQRT{} 약 0.707
SQRT2 : 2의 제곱근 SQRT{} 약 1.414

 

사용예) 반지름이 5인 원의 체적

<body>
<script>
    r = 5
    radia = r*r*Math.PI
    document.write(radia)
</script>
</body>

 

 

복잡한 계산을 풀어주는 Math 메소드

max(x,y) : x와 y의 수 중에서 가장 큰 수
min(x,y) : x와 y의 수 중에서 가장 작은 수
round(x) : x를 반올림한다
ceil(x) : x보다 같거나 큰 수 중에서 가장 큰 정수(올림)
floor(x) : x 보다 같거나 작은 수 중에서 가장 적은 정수(버림)
random() : 난수를 발생한다
abs(x) : 절대 값 함수 |x|

cos(x) : x의 cosine 값을 구해준다
sin(x) : x의 sine 값을 구한다
tan(x) : x의 tangent를 구한다
acos(x) : cosine의 역함수(arc cosine)
asin(x) : sin의 역함수(arc sin)
atan(x) : tangent의 역함수(arc tan)
atan2(x,y) : tangent의 역함수(arc tan)
exp(x) : x의 지수 함수를 구해준다
log(x) : x의 로그 함수를 구한다
pow(x,y) : 지수 함수 f(x,y) = xy
sqrt(x) : x의 제곱근을 구한다

 

사용예) rondom 메소드를 이용해서 원하는 숫자(작은 수에서 큰 수) 사이의 난수값을 도출하는 공식
: Math.floor(Math.random()*(큰수-작은수+1))+작은수
예를 들어 101 부터 130 까지의 난수를 생성하고 싶다면, Math.floor(Math.random() * (130-101+1)) +101
예를 들어 주사위처럼 1부터 6까지의 난수를 생성하고 싶다면, Math.floor(Math.random() * 6) +1

<script>
    var imgNum = Math.floor(Math.random()*3)+1
    document.write("<img src='img/img"+imgNum+".jpg' alt='랜덤이미지' />")
</script>

 

사용예) Math 메서소드 사용예

<script>
    r = 5.15
    document.write("x의절대값은",Math.abs(r),"입니다.","<br />")

</script>

 

 

 

JavaScript String 객체 또는 메소드, Array객체 또는 메소드

 

String(문자열) 객체

- String객체가 내장 객체 중에서 활용 빈도가 가장 높음

 

보편적인 생성법
- var  변수(인스턴트 네임) = new String("자바스크립트")

간편한 생성법
- var 변수(인스턴스 네임) = "자바스크립트"

var theText = "web js hi ciy"

theText.bold() : 텍스트를 진하게(강조) 표현하여 반환
theText.link('URL') : 텍스트에 링크를 걸어 반환
theText.length : 텍스트의 개수를 반환 (13)
theText.toLowerCse() : 영문자를 소문자로 바꿔 반환
theText.toUpperCase() : 영문자를 대문자로 바꿔 반환
theText.indexOf("i") : 처음부터 "i"를 찾아 최초로 일치하는 인덱스 번호를 반환, 없을 경우 -1을 반환 (8)
theText.lastIndexOf("i") : 끝에서부터 "i"를 찾아 최초로 일치하는 인덱스 번호를 반환, 없을 경우 -1을 반환 (11)
theText.charAt(5) : 5번째 인덱스에 저장된 문자 데이터를 반환 (s)
theText.substring(4,6) : 4번부터 6번 인덱스까지의 문자 데이터를 반환 (js)
theText.substr(10,3) : 10번 인덱스부터의 3글자를 반환 (ciy)
theText.replace("web","bye") : 텍스트 중 "web"을 "bye"로 치환해서 반환 (bye js hi ciy)
theText.slice(7,9) : 7번부터 9번 인덱스까지의 일부만 잘라서 반환 (hi)
theText.contcat("good") : 끝에 "good"이라는 문자를 결합하여 반환 (~good)
theText.split("") : (공백)을 기준으로 데이터를 분리(콤마) (web,js,hi,ciy)

 

사용예)

<script>
    document.write("<h3>문자치환</h3>")

    //3번 인덱스부터 끝까지 10.jpg
    theText2 = theText2.substring(3)
    //정수만 남기니깐 10

    document.write(theText2+"<br />")

    document.write("<h3>마지막 문자 추출</h3>")
    var theText3 = "hello javascript"
    var lastIndex = theText3.length-1
    theText3 = theText3.charAt(lastIndex)
    document.write(theText3+"<br />")

    document.write("<h3>특정 문자 포함여부</h3>")
    var theText4 = "tel phone now home"
    //indexOf 메서드는 찾는 문자가 없을 시 -1이란 값을 반환
    var findIndex = theText4.indexOf('mobile')
    if (findIndex<0) {
         document.write("찾는 문자가 없습니다.")
    } else {
        document.write("텍스트에 찾는 문자가 포함 되어있습니다.")
    }
</script>

배열 메서드

 

reverse() : 배열 데이터를 역순으로 반환

slice(1,2) : 인덱스 번호 1부터 시작하여 2직전의 데이터까지 잘라서 저장

sort() : 배열 데이터를 알파벳 혹은 0/1/2...의 오름차순으로 정렬 

join("구분자") : 배열 데이터를 구분자로 연결하여 결합   

concat() : 여러 개의 배열 데이터들을 하나로 결합

 

사용예)

<script>
    //배열 데이터 역순 정렬
    var theFood = ["짬뽕", "냉면", "갈비탕", "돈까스", "삼겹살"]
    theFood.reverse()
    console.log(theFood)

    //배열 데이터 오름차순 정렬
    var theSubway = ["9호선","2호선","7호선","4호선","3호선"]
    theSubway.sort()
    console.log(theSubway)

    //배열 데이터를 결합
    var Mountain1 = ["남산","도봉산","관악산","청계산","북한산"]
    var Mountain2 = ["설악산","지리산","한라산","속리산"]
    var joinResult = Mountain1.concat(Mountain2)
    console.log(joinResult)

    //원하는 데이터만 선택
    var theName = ["장준호","윤세정","홍우현","오윤석","박경훈"]
    var result = theName.slice(2,4)
    console.log(result)

    //구분자 기준으로 데이터를 합칩니다.
    var theProduct = ["냉장고","컴퓨터","선풍기","밥솥","핸드폰"]
    var joinResult = theProduct.join("-")
    console.log(joinResult)
</script>

 

 

배열 데이터 추가/삭제 메서드

 

shift() : 배열에서 첫 인덱스(0)에 저장된 데이터를 삭제

pop() : 배열에서 마지막 인덱스에 저장된 데이터를 삭제

unshift(값) : 배열에서 첫 인덱스(0)에 새 데이터를 중첩하여 추가

push(값) : 배열에서 마지막 인덱스에 새 데이터를 중첩하여 추가 

concat() : 여러 개의 배열 데이터들을 하나로 결합

 

사용예)

<script>
    var theSubway = ["2호선","9호선","7호선","4호선","3호선"]
    var newLine1 = "1호선"
    var newLine2 = "분당선"

    theSubway.shift()  //맨 앞(0번 인덱스)의 데이터 삭제
    //console.log(theSubway)     //콘솔 하나 씩 실행
    theSubway.unshift(newLine1)  //맨 앞(0번 인덱스)의 데이터 추가
    //console.log(theSubway)

    theSubway.pop()  //마지막(4번 인덱스)의 데이터 삭제
    //console.log(theSubway)
    theSubway.push(newLine2)  //마지막(4번 인덱스)의 데이터 추가(변경)
    //console.log(theSubway)
</script>
