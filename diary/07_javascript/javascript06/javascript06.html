내장 객체 사용하기

자바스크립트는 객체기반(object Based)의 스크립트 언어이다. '객체'란 '특정 기능'을 수행하는 단위체를 말하는데 우리의 평소 생활에서 예를 찾는다면, 평소 전화를 걸거나 받을 때 사용하는 '휴대폰'이나 혹은 어떤 물건을 멀리 보내야 할 때 이를 대행하는 '택배기사'와 '집배원' 등을 모두 '객체'라고 할 수 있다. 이때, 해당 객체가 지닌 각각의 기능을 자바스크립트에서는 '메서드'라는 용어로 정의한다.

  휴대폰.통화하기()  //메서드

  휴대폰.색상        //속성

 

자바스크립트는 항상 객체를 기반으로 기술하고, 객체명 뒤의 점(.)을 경계로 그 객체의 해당하는 메서드(기능) 또는 속성을 기술

- 자바스크립트에는 저마다 고유한 기능을 가진 다양한 종류의 객체들이 존재하는데, 크게 '내장 객체', '브라우저 객체','사용자 정의 객체'등이 있으며 '내장객체'는 자바스크립트 내에 내장된 객체로서 Date, Array, Math 등이 있으며, '브라우저 객체(BOM)'는 각 웹 브라우저에서 지원되는 객체로서 window, document(DOM), location 등이 있다.

 

사용예) Date객체 사용하기

<script>
    a=new Date()  //객체생성 : now연산자
    document.write(a)
</script>

 

 

메소드로 날짜와 시간 표시하기

  • getYear() 년도 (1900년 이후의 년도)
  • getFullYear() 년도를 구합니다
  • getMonth() 월을 구합니다
  • getDate() 일을 구합니다
  • getDay() 요일을 구합니다
  • getHours() 시간을 구합니다
  • getMinutes() 분을 구합니다
  • getSeconds() 초를 구합니다
  • getTime() 1900년 1월 1일 이후 시간을 1000분의 1초로 표시
  • getMilliseconds() 100분의 1초로 표시

 

사용예) 오늘 날짜 표시하기

<script>
    a=new Date()
    document.write("오늘은",a.getYear(),"년",a.getMonth()+1,"월",a.getDate(),"일","입니다.")
</script>

 

사용예) 요일 표시하기

<script>
    a=new Date()
    b=a.getDay()

    switch(b)
    {
        case 0 : c="일"
        break

        case 1 : c="월"
        break

        case 2 : c="화"
        break

        case 3 : c="수"
        break

        case 4 : c="목"
        break
        
        case 5 : c="금" 
        break
        
        case 6 : c="토" 
        break
      }

    document.write("오늘은 ",c,"요일 입니다.")
</script>

 

사용예) 특정 날짜 표시하기
a = new Date(년도,월,일,시간,분,초,1/100초)

<script>
    a=new Date(1991,11,18,11,32,11,11)
    document.write("내가 태어난 날짜는",a.getMonth(),"월",a.getDate(),"일",
                    a.getHours(),"시",a.getMinutes(),"분 입니다.")
</script>

 

사용예) 날짜 시간 변경하기

<script>
    a=new Date(1991,11,18,11,32,11,11)
    a.setDate(20)
    
    document.write("내가 태어난 날짜는",a.getMonth(),"월",a.getDate(),"일",
                    a.getHours(),"시",a.getMinutes(),"분 입니다.")
</script>

 

사용예) D-Day 표시하기

<head>
    <title>MY Birthday</title>
    <script>
        var today = new Date()
        var birth = new Date("June 1, 2002")
        // Janu Febr Marc Apri May June July Augu Sept Octo Nove Dece
        var aa = birth.getTime()
        var bb = today.getTime()
        var day = Math.ceil((bb-aa)/24/60/60/1000)
    </script>
</head>

<body>
    <img src="images/cup.gif" width="377" style="display: block; margin: 0 auto;" />

    <script>
        document.write("<p style='text-align: center;'>")
        document.write("내가 지구상에 태어난지 " + day + "일 지났어요.")
        document.write("</p>")
    </script>
</body>
